# 🗡️ Kimetsu no Yaiba - Demon Slayer Backend

Бэкенд-приложение на Django для управления вселенной **Клинка, рассекающего демонов**. Содержит информацию о персонажах (людях и демонах), их характеристиках и способностях.

---

## 📖 О проекте

Проект создан для фанатов аниме и манги **Demon Slayer**. Позволяет:
- Просматривать информацию о **столпах (Hashira)** и их дыханиях
- Изучать **демонов** и их способности
- Управлять данными через **Django Admin**
- Получать данные через **REST API**

---

## 🛠 Технологии

- **Python** 3.14
- **Django** 6.0
- **Django REST Framework** (для API)
- **SQLite** / PostgreSQL
- **Docker** (опционально)
